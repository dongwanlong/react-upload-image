LCE(Letv Compute Engine)云主机是什么
LCE提供基础资源可弹性伸缩的计算服务，资源的获取以及管理方式都优于传统物理服务器。用户可以根据业务的需求，随时获取和释放大规模云主机，使IT基础设施的使用及运维变得更简单、更高效，既帮助用户快速构建安全、可靠地应用环境，又能让用户将精力专注于核心业务。CPU、内存、磁盘等资源是LCE最基础的计算组件，LCE实例是云主机的最核心概念，也是呈献给每个用户的实际操作实体。
地域(region)
地域(region)指的是 LCE 实例所在的物理位置。可以根据您业务的分布和客户群体的分布在不同的Region内选择不同资源的LCE云主机，您可以同时拥有不同地域的LCE实例，目前我们仅提供北京地域资源供您选择，后续会根据需求不断建设或扩充其他地域资源。同一地域内的LCE实例内网间是可以互通的，不同地域之间的 LCE 实例内网不互通。
镜像
镜像是LCE实例运行环境的模板，包括操作系统和已安装的软件及配置信息。您可以使用镜像创建LCE实例。镜像分为两种：一种是乐视提供的“系统镜像”，为Linux操作系统。另一种是“自定义镜像”，由用户提供上传至乐视云计算平台，若要采用自定义镜像创建LCE实例，需预先与乐视服务人员联系上传相关事宜。系统镜像全局可见可用，自定义镜像只有用户本人可见可用。
存储
LCE提供了灵活、经济且易于使用的数据存储选项，这些存储选项既可以单独使用、也可以组合使用，以满足不同应用场景的需求。
云硬盘
云硬盘为云主机提供数据块级别的数据存储，它独立于云主机的生命周期而存在，可以被连接到任意运行中的云主机上，实现单独挂载，从而为云主机提供持久化的块级存储，采用三副本的分布式机制，提供 99.99999%的数据持久性保证。 目前乐视云仅提供容量型云硬盘，后续会提供性能型云硬盘以满足多业务运行需求。 性能型云硬盘：性能可达XXXMB/s，适用于对I/O要求较高的应用及I/O密集型应用，例如数据库等在线业务 容量型云硬盘：拥有更大的单块硬盘容积，最高可达2TB，适用于对容量要求较高的应用，例如文档存储等离线业务。I/O 性能：最高提供 3000 随机读写 IOPS 80MBps的吞吐性能；
快照
快照是指云主机或云硬盘在某一特定时间点的完整内容及结构的副本。快照的作用主要是在线数据备份与恢复，当设备发生应用故障或者文件损坏时，用户可以随时按快照点进行快速的数据恢复，从而实现包括增量备份等各种形式的备份服务。 有计划地对云主机和云硬盘创建快照，可保证业务可持续运行。
网络
基础网络与VPC
乐视云计算提供了两种组网方式：基础网络、私有网络（VPC）。 前者是一个由乐视云系统维护的全局网络，后者是用户根据业务需要自行组建的网络。 基础网络的优势就是简单、无需用户做任何配置与管理，即可直接使用，由于其是全局网络，因此安全由防火墙策略进行控制。与之相对应，私有网络需要用户创建并管理，私有网络（VPC）类似物理世界中使用交换机（L2 Switch）将多台服务器连接在一起， 组成的二层局域网，各个租户的私有网络是完全隔离，以满足用户对安全的100%追求。
公网IP
公网IP 地址是用户访问 LCE 实例以及 LCE实例对外提供服务的主要方式,公网 IP 地址是在互联网上合法的静态 IP 地址，并由乐视云计算统一分配。公网 IP 地址与您的账户而非特定的资源（主机或路由器）关联， 您可以将申请到的公网 IP 地址分配到任意位于基础网络或VPC中的资源，并随时可以解绑、再分配到其他资源，如此可以快速提供对外资源并实现资源的快速替换。
内网IP
每个LCE实例会分配一块私网网卡，并绑定一个内网 IP。内网 IP 是必选的且无法修改，如果您在操作系统内部自行变更内网 IP，会导致内网通讯中断。 同一地域内LCE实例之间通过内网 IP 进行的通讯，且流量是免费的。
路由器
作为公网与用户私有网络之间的通道 ，比如路由开启公网网关并绑定公网IP之后，便可以从VPC内部访问外部网络资源；如果开启公网网关，并给LCE实例绑定公网IP后，外部网络即可访问VPC内部资源。 默认的，只要子网关联在同一个路由器上，多个子网之间便自动打通了三层路由，可以彼此互通了。私有网络用户可以自行创建路由器，基础网络用户无法自动创建路由器。
